• GrowboxBackend 1.0

GreenMind API description

Resources

Authentication

Resource Description
POST /api/v1/login Authenticate and obtain a JWT token
POST /api/v1/logout Invalidate the JWT token and log out the user

Data

Resource Description
GET /api/v1/python_microservice/data/get_devices Get devices with caching
GET /api/v1/python_microservice/data/:device_id Get device data
POST /api/v1/python_microservice/data/update_device_data/:device_id Update device data
GET /api/v1/python_microservice/data/get_device_tasks/:device_id Get device tasks
GET /api/v1/python_microservice/data/get_device_data_history/:device_id Get device data history
POST /api/v1/python_microservice/data/add_device_task/:device_id Add device task
PUT /api/v1/python_microservice/data/update_device_task/:device_id Update device task
POST /api/v1/python_microservice/data/add_device Add device
GET /api/v1/python_microservice/data/get_data Get microservice data
GET /api/v1/python_microservice/data/get_tasks Get microservice tasks

Device_logs

Resource Description
GET /api/v1/device_logs Get all device logs for a specific device
GET /api/v1/device_logs/:id Show details of a specific device log

Devices

Resource Description
GET /api/v1/devices Get all devices of the current user
GET /api/v1/devices/:id Show details of a specific device
DELETE /api/v1/devices/:id Delete a device
POST /api/v1/devices/assign Assign a device to the current user
PUT /api/v1/devices/:id Update a device
POST /api/v1/devices Create a new device
GET /api/v1/devices/:id/plants Get all plants associated with a specific device

Plants

Resource Description
GET /api/v1/plants/public_index Get all public plants
GET /api/v1/plants/:id Show details of a plant
GET /api/v1/plants/private_index Get all private plants of the current user
GET /api/v1/plants/private_show/:id Show details of a private plant
GET /api/v1/plants/assigned_index Get all assigned plants of the current user
POST /api/v1/plants Create a new plant
PUT /api/v1/plants/:id Update a plant
DELETE /api/v1/plants/:id Delete a plant
POST /api/v1/plants/add_to_favorites Add a plant to favorites
POST /api/v1/plants/remove_from_favorites Remove a plant from favorites
GET /api/v1/plants/favorite_plants Get all favorite plants of the current user

Users

Resource Description
GET /api/v1/users Get all users
GET /api/v1/users/:id Show current user
GET /api/v1/users/show_code Show code for the current user
GET /api/v1/auth_code?code=<code> Authenticate user by code
POST /api/v1/users Create an user
